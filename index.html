História dos computadores

A história da evolução dos computadores pode se dizer que começou no início do século XIX(19), com a invenção de um tear que utilizava cartões perfurados para tecer automaticamente, criado por Joseph Marie Jacquard.

Em 1822, o inglês Charles Babbage projetou uma máquina analítica, que seria capaz de calcular tabelas de números. Infelizmente o projeto foi um fracasso devido a falta de financiamento e complexidade.

Temos também, Herman Hollerith, que contribuiu para a história, com um sistema de cartões perfurados para calcular o censo de 1880, que forneceu o resultado em 3 anos, economizando muitos anos de trabalho. Mais tarde, ele estabeleceu uma empresa que acabaria se tornando a IBM

Dando um salto no tempo, temos o Harvard Mark 1, sendo a primeira máquina operacional que executava cálculos longos de forma automatica, projetado por Howard Aiken e construído pela IBM.
O Computador e Integrador Numérico Eletrônico, conhecido como ENIAC, foi de fato o primeiro "computador", sendo desenvolvido a pedido do exército dos Estados Unidos para o laboratório de pesquisa balística. Ele computava trajetórias táticas que exigiam conhecimento em matemática de forma ágil, e era operado a partir de cartões perfurados manualmente por funcionários do exército, e para conseguir determinar a tarefa que seria realizada, era necessário conectar fios, relês e sequências de chaves, e para cada tarefa diferente seria preciso refazer todo o processo.
Tanto o Harvard Mark 1 e o ENIAC são considerados computadores de 1o geração.

Já a segunda geração, é marcada pelo uso de transistores, em vez de válvulas eletrônicas, permitindo que computadores consumissem menos energia e tivessem seu tamanho reduzido. Em 1960, foi criado os circuitos integrados, que juntou em um único dispositivo, centenas de transistores, possibilitando computadores ainda menores e mais rápidos

O que são Clients?

Podemos definir Clients como os consumidores, alguém que irá consumir determinado produto, como por exemplo usuários de redes sociais, utilizando algum dispositivo para poder acessar tal serviço.

O que são Servers?

Servers são conhecidos por hospedar os produtos e executar eles, sendo mais eficiente que um computador pessoal. E também há diversos tipos de servidores, como por exemplo servidor de banco de dados que podem guardar informações, servidor de mensagem eletrônica, entre outros.

Como é uma estrutura básica de um HTML?

Podemos comparar a estrutura básica de um HTML com a anatomia do corpo humano, tendo cabeça (head) e corpo (body), na cabeça do html colocamos informações para o navegador interpretar, como a mensagem de título, dados sobre a página, e escala da interface. E o corpo do HTML é onde fica de fato o que será apresentado pela página, tendo as tags para definir exatamente o que queremos colocar em evidência.

Entendendo a diferença entre listas ordenadas e não ordenadas

Dentro do HTML temos as tags de listas ordenadas e não-ordenadas, que fazem jus ao nome, servindo para fazer listas dessa forma:

lista ordenada:

  1. item
  2. item
  3. item
lista não-ordenada:
  • item
  • item
  • item


Na lista ordenada, podemos observar que ela segue uma numeração, como se fosse uma lista de passos em ordem, já na lista não-ordenada, ela é representada por pontos, que podem ser alterado via style css, que não remete a uma ordem, mas podemos exemplificar como uma lista de de necessidade, que não precisa seguir uma ordem.
